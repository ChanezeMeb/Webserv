🔄 Test des Redirections HTTP

301 - Moved Permanently (Redirection permanente)

Le navigateur mémorise cette redirection.

Test : Aller vers /old-page

Commande curl :

curl -v http://localhost:8080/old-page

Attendu : HTTP/1.1 301 Moved Permanently + Location: /new-page

302 - Found (Redirection temporaire)

Le navigateur ne mémorise pas cette redirection.

Test : Aller vers /temp-redirect

Commande curl :

curl -v http://localhost:8080/temp-redirect

Attendu : HTTP/1.1 302 Found + Location: /

Redirection externe

Redirige vers un site externe.

Test : Aller vers /google

Commande curl :

curl -v http://localhost:8080/google

Attendu : HTTP/1.1 302 Found + Location: https://www.google.com

✅ Comment vérifier que ça fonctionne ?

Avec curl, tu dois voir :

< HTTP/1.1 301 Moved Permanently < Location: /new-page < Content-Type: text/html

Le header Location: indique où le navigateur doit aller.
